This package contains ImageIO based raster reader and writer.

The supported image types are determined at runtime by IIORasterIOProvider. It should support at least JPEG, PNG and GIF. (GIF read only). With the JAI ImageIO Tools it should support JPEG, JPEG 2000, PNG, GIF and TIFF.

IIORasterReader is able to read GeoTIFF metadata like resolution, crs and bounding box.
