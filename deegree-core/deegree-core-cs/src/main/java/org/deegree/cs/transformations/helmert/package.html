A Transformation can be defined as an mathematical operation which transforms some values into other values using a specified function. In geographic terms this means, that some incoming ordinate in a specified crs, must be transformed into a ordinate valid for some other crs.

The helmert transformation allows coordinate transformation with a datum shift by going to euclidean space and normalizing to WGS84, this normalizing is done using a helmert transformation.
